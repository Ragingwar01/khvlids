Basic Kali Linux Tools

  • Nmap: Used for network discovery and security auditing.

    Command: nmap -A target-ip

    Scans a target for open ports and running services.

  • Metasploit: A powerful framework for penetration testing.

    Command: msfconsole

    Launches the Metasploit framework for exploiting vulnerabilities.

  • Aircrack-ng: Used for assessing Wi-Fi network security.

    Command: airmon-ng start wlan0

    Enables monitor mode to capture Wi-Fi packets for analysis.

  • John the Ripper: A fast password cracker.

    Command: john --wordlist=passwords.txt hashes.txt

    Uses a wordlist to crack hashed passwords.

  • Burp Suite: A tool for web vulnerability scanning.

    Command: burpsuite

    Opens the Burp Suite GUI for intercepting and analyzing web traffic.
